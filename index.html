javatari

A multiplayer Atari 2600 emulator written in pure Java with no external libs.

Download this project as a .zip file Download this project as a tar.gz file

Download the executable JAR here.

Needs Java 6 or greater and Atari ROMs.

Features

  • Unique Client-Server multiplayer mode. Runs great in low-latency networks such as LANs.
  • Complete Save State/Load State functions.
  • Scanlines and TV screen emulation modes.
  • Sizable screen and fullscreen mode with Vsynch.

Controls

P1 Controls: Arrow Keys / Space: Fire
P2 Controls: F,T,G,H / A: Fire
Alt + J: Swap P1<>P2 Controls
Alt + L: Connect/Disconnect Paddles

Alt + ENTER: Toggle FullScreen
Alt + S: Toggle Scanlines
Alt + Q: Toggle Filter
Alt + D: Toggle Debug mode
Alt + C: Toggle Collisions
Alt + V: Toggle NTSC/PAL
Alt + P: Toggle Pause

Ctr + 1-9: Save State
Alt + 1-9: Load State

Alt + Arrows: Screen Size
Backspace: Default Screen Size

TAB: Fast Speed
